# Agar.io

## Test Task 1:
### Додано кнопку "Exit"
- На клієнті (`index.html`) додано кнопку для виходу з гри.
- **Використано бібліотеки:**
  - SweetAlert2 для інтерактивних модальних вікон.
  - Font Awesome для іконки дверей.

### Реалізовано механізм виходу з гри:
- При натисканні на кнопку "Exit" з’являється попереджувальне модальне вікно:
  - Користувач може підтвердити або скасувати вихід.
- Якщо користувач підтверджує вихід:
  - Відображається таймер на 10 секунд із зворотним відліком.
  - Після завершення таймера виконується перенаправлення на головну сторінку (сторінка оновлюється).

## Test Task 2:
### Новий тип їжі — Rare Food
- У функції `spawnFood` додана логіка для рідкісного спауну Rare Food. Можна встановити власний відсоток респауну у співвідношенні із звичайною їжею.
- При споживанні цього типу їжі розмір клітини подвоюється.
- Додано обмеження щодо максимальних розмірів моделі після поїдання Rare Food.

### Особливості:
- **Тип їжі (type):** 4.
- **Колір:** золотий (`r: 255, g: 215, b: 0`) для відмінності від звичайної їжі.
- **Примітка:** В подальшому можна замінити на іншу геометричну фігуру для більшої візуальної різноманітності.
